 Documentation Convention

This documentation follows the text conventions as described below.

External applications are indicated in italics, Helvatica, deep-red such as: Tracker.

Names of files, paths etc. are formatted in Teletype in a green such as: /boot/apps or Beezer.pkg etc.

Keyboard strokes, combinations are indicated in bold, Teletype, purple such as: the SHIFT key!

Menu items, toolbar buttons preference items are formatted using Helvetica, navy such as: Actions –> Extract or toolbar: Extract etc. This means to choose the "Extract" command under the "Actions" menu or to click the "Extract" toolbar button from the toolbar.
