Installing on BeOS

Beezer is distributed as a SoftwareValet package. Just locate the file Beezer.pkg and double-click it. Now Beezer's installer will begin. Follow the instructions on screen and complete the installation.

You can move Beezer to another location even after installation, as all of its application and data files are self-contained in the Beezer folder. You can move the folder to any BeOS File System as long as you keep the data files within it and don't alter the directory structure.

Important note:

No matter to which folder you install or move Beezer, that folder must be inside a BeOS File System (BFS volume). Some of Beezer's operations require BFS support, hence you cannot install Beezer to a FAT16 or FAT32 partition.
