 Deleting archives

Beezer allows you to delete the archive you have currently opened. Once you choose to delete the archive from File –> Delete, you will be prompted whether to delete permanetly or move the archive to the system Trash or cancel the delete.

Note: If you delete an archive permanently you will NOT be able to recover the archive. Use this option carefully.
